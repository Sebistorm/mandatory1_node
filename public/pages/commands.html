Mandatory 1 - Commands

npm install

npm install installs a package and any packages that it depends on. If the package has a package-lock, the installation of dependencies will be driven by that. It will install the dependencies to the local node_modules folder

npm install

npm install package_name

npm install package_name is used to install a package and add the dependency to the package.json file. The following way is to install Express to the application:

npm install express

npm init

npm init creates a package.json file where the name of the project, version, repository, description, etc. can be specified.

npm init

package.json

package.json is used to store meta data (name of the project, version etc.), dependencies, etc. The following code is a short version of a package.json file:

"name": "mandatory_node", "version": "1.0.0", "description": "Mandatory 1 nodejs", "dependencies": { "express": "^4.17.3" }

git commands

The most important git commands:

git clone repository url // cloning the repository into the folder git add -A // adds every files that has changes git commit -m "insert message here" // makes a commit with a message git push // pushing all the commits to the remote repository git pull // pulls the code from the remote repository to the local repository if it detects changes
