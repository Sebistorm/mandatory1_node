Mandatory 1 - Nodemon

Installing Nodemon

Nodemon is a tool that helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected. The recommended way is to install it globally by doing this command:

npm install -g nodemon

nodemon.json

nodemon.json can be used to specify which type of files nodemon should detect changes on. It can be done the following way:

{ "ext": "js, json, html" }

Starting nodemon

The nodemon tool can be executed with the following commands:

nodemon file_name
