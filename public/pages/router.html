Mandatory 1 - Router

Router

The router is used to define the routes then use it in an Express application. The following code shows a basic example of code inside the router file:

const router = require("express").Router(); router.get("/", function (req, res) { res.send("About this wiki"); }) module.exports = router;

Using the router file inside app.js

Using the route file can be done the following way:

const wikisRouter = require("./routers/wikisrouter.js"); app.use(wikisRouter);
